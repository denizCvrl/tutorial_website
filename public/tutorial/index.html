Making a Website with R and the Blogdown Package

Prerequisites ðŸ˜„

  • Being able to install a program in your computer
  • Being patient and determined

The tutorial benefits from the extensive documentation of the blogdown package and the academic theme but it doesn't explain every single option. It just adapts one of the methods suggested in these documents.

Lesson 1 - Install Blogdown

You need both R and RStudio IDE. R is a programming language for statistical computing and graphics while R Studio is an integrated development environment (IDE) that provides a user-friendly platform.

Lesson 2 - A Quick Start

In this lesson, you will build the example site of the academic theme. Later, we will modify the elements of this website. Our aim is to teach you the structure of folders so that you can apply a similar logic while using other themes.

Lesson 3 - Menu of Your Website

In this lesson, there is a short discussion of “projects” in R. Then, we will be back to working on the website to learn how to create the menu of your website.

Lesson 4 - Themes, Colors, Fonts

In the academic theme, you have the flexibility of creating your own color and font theme. Follow along to learn more. params.toml is like your make-up kit, but instead of blush and mascara, you have theme, color, font, fontsize, etc.

WIDGETS ðŸŽ¯ are built-in designs for your website. Not every theme has widgets, but the academic theme has lots of them. We will look at them one by one and discuss how you can modify them in your website.

More WIDGETS ðŸŽ¯ In this lesson we will cover all the widgets that appear on the homepage of the example webpage. All you need to do is to decide which ones you would like on your own webpage.
